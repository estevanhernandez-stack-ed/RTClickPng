Transitive dependencies
---------------------------------------------------------------------------

libheif pulls in libde265 (LGPL-3.0) — same compliance path as libheif above.
libavif pulls in aom (BSD-2-Clause) or dav1d (BSD-2-Clause) depending on
vcpkg feature selection at build time. The installed component list for
the current build is captured in build/vcpkg/installed/x64-windows/share/
after `vcpkg install` runs.
